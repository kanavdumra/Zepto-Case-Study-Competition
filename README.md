# Zepto Case Study Competition

A product management case study focused on improving Zepto's Search-to-Cart conversion through data-driven analysis and UX improvements.

## Features
- User journey and funnel analysis
- Root cause identification using a Priority Matrix
- UX improvement recommendations
- KPI definition and A/B testing framework

## Tools Used
- Microsoft Excel
- PowerPoint
- Product Analytics Concepts

## Outcome
Proposed actionable strategies to improve user experience and increase conversion rates.
